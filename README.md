# TeamSpirit Worktime Filler

TeamSpirit の勤務表ページで、編集可能な勤務日に以下を入力する Chrome 拡張です。

- 出社: `10:00`
- 退社: `18:30`
- 工数: `Dev` に `100%`

月次の承認申請は実行しません。

## インストール

1. Chrome で `chrome://extensions` を開く。
2. 右上の「デベロッパー モード」を有効化する。
3. 「パッケージ化されていない拡張機能を読み込む」を押す。
4. このディレクトリを選択する。

```text
/Users/hhoriuchi/go/src/github.com/i-eight/anima/console/chrome-extension
```

## 使い方

1. TeamSpirit の勤務表ページを開く。
2. 対象月を勤務表上で選択する。
3. 拡張機能のポップアップを開き、「勤務表を入力」を押す。
4. 既存入力がある場合は確認ダイアログが出る。キャンセルすると何もしない。

## 対象行

勤務表上の平日行を対象にします。土日は除外します。出社/退社ダイアログを開いた時に入力欄が無効または読み取り専用の場合、その日はスキップします。

## 注意

- TeamSpirit の画面 DOM に依存しています。画面更新で id や構造が変わると修正が必要です。
- 日別の「登録」ボタンは押しますが、月次の「承認申請」は押しません。
